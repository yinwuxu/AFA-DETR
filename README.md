# AFA-DETR
Our Paper：AFA-DETR: Asymptotic Frequency-Attention DETR with Spectral-Spatial Hybrid Encoder for fabric Defect Detection 



ABSTRACT
Transformer-based object detection methods have demonstrated superior performance in general vision tasks but encounter severe representation bottlenecks in industrial fabric defect detection. The periodic fabric texture generates multi-band noise, resulting in spectral overlap between the high frequency components of subtle defects and the background, while macro-scale anomalies necessitate extensive global context modeling. Furthermore, significant intra-class scale variations and low contrast characteristics further constrain the representational capacity of conventional methods. To overcome these limitations, this paper proposes AFA-DETR, a progressive detection framework based on spatial-frequency collaborative perception. Specifically, to address background interference and spectral overlap, a Global Frequency Token Mixer (GFTM) is designed to implement spectral dynamic cross-correlation via 2D Fast Fourier Transform (FFT), achieving adaptive decoupling of background noise with minimal computational overhead. For scale variations, a Residual-enhanced Ghost Progressive Adjacent Feature Fusion (RG-PAFF) module is constructed to strengthen context information flow through multi-scale progressive interaction logic. Regarding the low-contrast challenge, a parameter-free Contrast-sensitive Attention Mechanism (C-SimAM) is introduced, which significantly enhances the identification of subtle defects by preserving linear residual polarities.Experimental results on a custom dataset show that AFA-DETR achieves 96.8% mAP50 and 64.5% mAP with only 8.7M parameters and 14.3GFLOPs (representing a 42.3% reduction in computational overhead). Cross-architecture generalization experiments validate the plug-and-play capability of the model, establishing a new paradigm for resource-constrained industrial quality inspection. The source code and dataset are publicly available at https://github.com/yinwuxu/AFA-DETR.
